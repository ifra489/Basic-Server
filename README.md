# 🚀 Basic Node.js HTTP Server

This project demonstrates how to create a simple HTTP server using Node.js.

The server listens for incoming requests and responds with plain text.

---

## 📌 Features

- Create HTTP server using built-in http module
- Handles request-response cycle
- Sends status codes and headers
- Runs on custom port

---

## 🛠️ Technologies Used

- Node.js
- HTTP module

---

## ▶️ How to Run Locally

1. Clone the repository:

git clone YOUR_REPO_LINK

2. Install dependencies (if any):

npm install

3. Start the server:

npm start

4. Open browser:

http://localhost:5000
